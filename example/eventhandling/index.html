This is an example of event handling. Cornerstone fires the event 'CornerstoneViewportUpdated" any time a viewport property is changed. Listen for this event and the event passed to your callback function will include the element and the updated viewport. Cornerstone first the event 'CornerstoneImageRendered' everytime the image is updated. The event includes the canvas context set to the image coordinate system so you can draw overlays.

This example illustrates this by updating the bottom left overlay with the updated WW/WC values. It also illustrates overlays on the image itself by drawing a rectangle around a tumor and labeling it with the text "Tumor Here"
